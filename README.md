# 🌍 Centros Espíritas Próximos

Aplicativo mobile desenvolvido com **React Native + Expo** que utiliza geolocalização para encontrar centros espíritas próximos ao usuário, exibindo informações úteis e facilitando o acesso via Google Maps.

---

## 🚀 Sobre o projeto

Este projeto foi criado com o objetivo de:

- Aplicar conceitos de geolocalização em apps mobile
- Integrar APIs externas (Google Places e Maps)
- Criar uma experiência simples, intuitiva e útil
- Explorar monetização com AdMob
- Desenvolver um produto real para portfólio

---

## 📱 Funcionalidades

- 📍 Detecção automática da localização do usuário
- 🔎 Busca de centros espíritas próximos
- 🗺️ Visualização no mapa
- 📏 Cálculo de distância
- 📄 Tela de detalhes com:
  - Endereço
  - Telefone
  - Site
  - Horários (traduzidos para português)
- 🧭 Botão para abrir rota no Google Maps
- 🚫 Filtro automático para ocultar resultados indesejados
- 📢 Banner de anúncios (AdMob)
- ⏳ Loading com estados amigáveis
- ❌ Tratamento de erros e lista vazia

---

## 🛠️ Tecnologias utilizadas

- React Native
- Expo
- TypeScript
- React Navigation
- Expo Location
- Google Places API
- Google Geocoding API
- React Native Maps
- AdMob (react-native-google-mobile-ads)
- EAS Build
